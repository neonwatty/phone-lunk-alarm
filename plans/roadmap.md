## Planned Phases

### Phase 2: Engagement & Virality
**Goal:** Increase sharing and user-generated content

- [ ] **Leaderboard** - Track "lunks caught" with anonymous stats
- [ ] **Custom text overlays** - Let users add their own captions to recordings
- [ ] **Instagram Reels export** - Optimize for IG format alongside TikTok
- [ ] **Watermark toggle** - Optional @phonelunk watermark on exports
- [ ] **Sound upload** - Let users upload their own alarm sounds

### Phase 3: Gym Integration
**Goal:** Partner with gyms for official deployments

- [ ] **Kiosk mode** - Full-screen view for gym TVs
- [ ] **Multi-camera support** - Switch between camera feeds
- [ ] **Admin dashboard** - Stats, settings, moderation
- [ ] **API endpoints** - For gym management system integration
- [ ] **Privacy mode** - Blur faces option for public deployments

### Phase 4: Monetization
**Goal:** Sustainable revenue model

- [ ] **Premium sounds pack** - Additional alarm sound options
- [ ] **Custom branding** - White-label for gyms
- [ ] **Analytics dashboard** - Advanced stats for gym owners
- [ ] **Ad-supported free tier** - Optional ads in free version

### Phase 5: Mobile Apps
**Goal:** Native experience on iOS/Android

- [ ] **PWA improvements** - Better offline support, install prompts
- [ ] **React Native app** - Native iOS/Android apps
- [ ] **Push notifications** - Alert when lunk detected
- [ ] **Background detection** - Run while phone is locked
